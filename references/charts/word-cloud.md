# 词云（Word Cloud）

## Overview

词云用于**文本数据可视化**——把一组「词 + 权重」可视化为一片文字，**字号大小映射权重**。常用于关键词分析、热点话题、评论情感等场景。

典型应用：

- 关键词热度（搜索词、商品标签）
- 评论 / 文本主题分析
- 财经新闻热点提取

---

## 数据结构

两种形式：

**无坐标轴（普通词云）**：每条数据是 `[word, weight]`

```
[
  ['股票', 100],
  ['基金', 80],
  ['债券', 50],
  ...
]
```

**单轴词云**（沿时间 / 类别轴排列）：每条数据是 `[axisValue, word, weight]`

```
[
  ['2024-Q1', '股票', 100],
  ['2024-Q1', '基金', 80],
  ['2024-Q2', '股票', 90],
  ...
]
```

---

## 坐标系

| 模式 | 说明 |
| --- | --- |
| `none`（默认） | 无坐标轴，词在画布内自由布局 |
| `singleAxis` | 单轴布局，词沿时间 / 类别轴分布 |

---

## 字号映射

字号按权重映射到一个范围内：

| 配置 | 默认 |
| --- | --- |
| 字号范围 | **`[12, 24]`** px |
| 字号缩放函数 | `linear`（默认） / `sqrt` / `log` / 自定义函数 |

- `linear`：线性映射（权重高的词字号大）
- `sqrt`：平方根映射（极值压缩，避免最大词过大）
- `log`：对数映射（权重分布跨度大时用）

---

## 字重映射

字重也可按权重映射：

| 配置 | 默认 |
| --- | --- |
| 字重范围 | **`[300, 600]`** |
| 字重缩放 | `linear` / `sqrt` / `log` |

权重高的词不仅字号大，字重也更粗。

---

## 布局

### 普通词云（`coordinateSystem: 'none'`）

| 配置 | 说明 |
| --- | --- |
| 精度（`precision`） | 布局精度（越大越不精确，性能越好；默认 2） |
| 重叠阈值（`overlapAreaThreshold`） | 平均重叠面积低于阈值时仍允许显示（避免过严布局导致词无法放入；默认 10） |
| 文字间距（`padding`） | 文字与图形间距（默认 1） |
| 图形间距（`margin`） | 文字 / 图形之间间距（默认 0） |
| 旋转范围 | `[0, 2π]`（弧度）；可控制词旋转角度的随机范围 |

### 单轴词云（`coordinateSystem: 'singleAxis'`）

| 配置 | 说明 |
| --- | --- |
| 模糊宽度 | 模糊布局的横向范围（最小 0） |
| 边缘对齐 | 主轴两侧超出 grid 的词向内收敛（可设阈值） |
| 聚合（`aggregation`） | 单词聚合，重叠比例 0-1（默认 0.3） |

---

## 蒙版（Mask，特殊形态）

可指定一个形状 / 图片作为蒙版，让词在蒙版形状内填充：

| 模式 | 说明 |
| --- | --- |
| `contain` | 仅渲染一次（默认） |
| `cover` | 多次渲染填充容器 |
| `fill` | 等比拉伸图形填充容器 |

---

## 层级（z-index）

词云默认 **`z: 999`** —— 在所有图表元素之上层，确保不被遮挡。这是**图表级覆盖**：压过 base 基线最高层（轴标签 57），属 [base.md § Z-Index 层级 — 覆盖规则](../themes/base.md#覆盖规则) 允许的图表级专属层级声明。

## 实现要点（库无关）

- **字号范围控制视觉差**：太小看不清（< 12），太大占满画布（> 32）；默认 `[12, 24]` 适合大部分场景。
- **缩放函数按数据分布选**：权重均匀用 `linear`；权重悬殊用 `sqrt` 或 `log` 压缩极值。
- **不要全词显示**：词太多会糊成一团，按权重取 top N（如 50-100 词）。
- **布局精度 vs 性能**：精度高布局美观但慢，移动端可降低精度。
- **旋转角度别太多**：> 2 种角度（如 0° + 90°）容易乱；保持 1-2 种角度为佳。

---

## Do & Don't

| | 规则 |
| --- | --- |
| ✅ | 字号映射到 `[12, 24]` 范围（默认） |
| ✅ | 字号 + 字重双重映射，权重感更强 |
| ✅ | 权重分布悬殊时用 `sqrt` / `log` 压缩极值 |
| ✅ | 词数控制在 top 50-100，按权重取 |
| ❌ | 不要让字号 < 12px（看不清） |
| ❌ | 不要无限制添加词数（> 200 必糊） |
| ❌ | 不要用 3+ 种旋转角度（视觉混乱） |
