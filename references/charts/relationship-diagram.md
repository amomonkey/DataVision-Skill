# 关系图谱（Relationship Diagram）

## Overview

关系图谱用于展示**多个实体之间的层级 / 网络关系**——节点表示实体，连线表示关系。paradigm-chart 的实现是**树状关系图谱**（一棵或多棵树并列），节点按深度分级排列。

典型应用：

- 股权关系图（公司 → 股东 → 上层股东）
- 产品 / 业务关联图谱
- 组织 / 人员关系网络

---

## 数据结构

每个节点是一个树形对象，多棵树用数组并列：

```
[
  {
    name: '根节点 A',
    priority: 1,
    backgroundColor: ['#xxx', '#yyy'],
    children: [
      { name: '子节点 A1', collapsed: true, children: [...] },
      { name: '子节点 A2', ... }
    ]
  },
  { name: '根节点 B', children: [...] }
]
```

节点属性：

| 属性 | 说明 |
| --- | --- |
| `name` | 节点名 |
| `priority` | 优先级（影响布局排序） |
| `backgroundColor` | 节点背景色（支持渐变数组） |
| `watermark` | 节点上的水印文字 |
| `render` | 自定义渲染配置（文字 / 附件 / logo） |
| `children` | 子节点 |
| `collapsed` | 是否折叠（true 时初始折叠） |
| `parentId` | 父节点 id（跨树关联用） |

---

## 布局

| 配置 | 说明 |
| --- | --- |
| 初始展开深度（`initialTreeDepth`） | 根深度为 0，展开到第几层 |
| 全局折叠（`collapsed`） | 整张图初始折叠状态 |
| 节点竖直间距（`dvVerticalSpacingBetweenNode`） | 数组，按层级配置每级垂直间距 |
| 首屏自适应（`dvAdapt`） | 默认 `true`，自动缩放适配视口 |
| 节点间距函数（`dvNodeGap`） | 自定义函数控制节点间距 |

---

## 节点渲染

节点可完全自定义渲染：

- 默认：方框 + 文字 + 可选 logo / 附件
- `dvNodeRender`：自定义函数返回 SVG 图形组
- 支持渐变背景、阴影、圆角等

---

## 连线

| 类型 | 配置 |
| --- | --- |
| 父子连线 | 默认显示，连接父节点与子节点 |
| 同级分支线（`dvLevelLine`） | 同一深度的兄弟节点之间可加分支线 |
| 跨树导向线（`dvToNextTreeLine`） | 多棵树时，可在树之间加导向线 + 箭头 |

**导向线默认样式**：

| 属性 | 默认 |
| --- | --- |
| 线宽 | 0.5px |
| 颜色 | `rgba(0,0,0,0.24)` |
| 箭头线宽 | 1px |
| 箭头颜色 | `rgba(0,0,0,0.24)` |
| 箭头缩放 | 1 |

---

## 水印（`dvWatermark`）

可在节点上或图表整体加水印（数组形式，支持多个文字水印）。

---

## 坐标系

`view` 坐标系（自由布局，非笛卡尔轴）。

---

## 实现要点（库无关）

- **多树并列时用导向线**：单棵树用普通父子连线；多棵树（如多家公司股权对比）用跨树导向线 + 箭头明示流向。
- **初始展开深度控制密度**：层级深、节点多时默认展开 1-2 层，深层按需点开，避免一次性渲染整张图谱。
- **节点优先级排序**：同级节点按 `priority` 排序，重要节点（核心实体）在视觉中心。
- **背景色 / 水印做实体分类**：用不同背景色 / 水印区分实体类型（个人 / 公司 / 机构）。
- **首屏自适应必开**：图谱节点位置不可预知，必须 `dvAdapt: true` 自动缩放适配。

---

## Do & Don't

| | 规则 |
| --- | --- |
| ✅ | 多棵树用导向线 + 箭头明示跨树关系 |
| ✅ | 初始展开 1-2 层，深层折叠 |
| ✅ | 用背景色 / 水印区分实体类型 |
| ✅ | 启用首屏自适应缩放 |
| ❌ | 不要默认全展开——深层图谱视觉爆炸 |
| ❌ | 不要让导向线和父子连线同色——会视觉混淆 |
| ❌ | 不要用关系图谱展示纯层级数据（那是双向树图的场景） |
