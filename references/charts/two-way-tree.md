# 双向树图（Two-Way Tree）

## Overview

双向树图用于展示**层级结构**——根节点居中，**向两个方向对称展开**子节点。常用于组织架构、家谱、产品分类、决策树等场景。

与普通树图的区别：普通树图单方向展开；双向树图根节点居中，左右（或上下）对称分叉。

---

## 数据结构

树形递归：

```
{
  name,
  children: [
    { name, children: [...] },
    { name, children: [...] }
  ]
}
```

节点支持自定义样式：`itemStyle` / `lineStyle` / `label` / `symbol` / `symbolSize`。

---

## 布局方向

| 布局 | 走向 |
| --- | --- |
| `horizontal` | 水平展开（`LR` 从左到右 / `RL` 从右到左） |
| `vertical` | 垂直展开（`TB` 从上到下 / `BT` 从下到上） |

---

## 连线形态

| 类型 | 说明 |
| --- | --- |
| 曲线（`curve`） | 节点间用平滑贝塞尔曲线连接（默认） |
| 折线（`polyline`） | 直角折线连接 |

连线两端可加箭头 / 圆点等符号。

---

## 展开 / 折叠

| 能力 | 说明 |
| --- | --- |
| 初始展开深度 | 配置默认展开到第几层（如展开 2 层） |
| 点击展开 | 点击节点展开 / 折叠下级 |
| 折叠状态 | 由数据节点的 `collapsed` 字段控制 |

---

## 层级样式

可对不同深度的节点 / 连线分别配置样式（按 `levels` 数组），实现"根节点大、叶子节点小"等视觉层次。

---

## 节点样式

| 元素 | 默认 |
| --- | --- |
| 符号 | 圆形（可换其他符号） |
| 大小 | 15px（默认） |
| 颜色 | 系列色 |
| 标签位置 | 节点旁（`top` / `right` 等） |

---

## 动画

- 入场动画时长：默认 **550ms**
- 缓动：`linear` / `cubicOut` 等
- 展开 / 折叠时连线 / 节点同步动画

---

## 实现要点（库无关）

- **根节点居中**：双向树的关键是"对称展开"——根节点必须居中，左右（或上下）均衡延伸。
- **连线形态影响阅读**：层级深 / 节点多用曲线（柔和、不挤压）；结构严谨用折线（规整、对齐）。
- **初始展开深度控制视觉密度**：全展开易糊，默认展开 2 层，深层按需点开。
- **层级样式区分主次**：用 `levels` 给不同深度配不同节点 / 连线样式，主线突出、细枝弱化。

---

## Do & Don't

| | 规则 |
| --- | --- |
| ✅ | 根节点居中，左右 / 上下均衡展开 |
| ✅ | 初始展开 2-3 层，深层按需展开 |
| ✅ | 用层级样式区分主次（根节点大、叶子小） |
| ✅ | 节点多用曲线连线（柔和），结构严谨用折线 |
| ❌ | 不要默认全展开——层级深时视觉爆炸 |
| ❌ | 不要让连线穿越节点——会误导阅读 |
| ❌ | 不要用双向树展示无层级关系的数据（那是关系图谱的场景） |
