# 排名线图（Rank Line）

## Overview

排名线图（Rank Line）用于展示**多个实体在时间维度上的排名变化**——每条线代表一个实体，Y 轴表示排名（通常 1 在顶部），X 轴是时间。线条的起伏直观呈现排名升降。

典型应用：

- 球队 / 选手赛季排名变化
- 公司市值 / 销量排名演变
- 商品热度 / 关键词搜索排名

排名线图是基础折线图的扩展形态——技术上基于 ECharts line series 增强。

---

## 数据结构

类折线图的数据结构，但 Y 值是「排名」而非「数值」：

```
[
  { name: '球队 A', data: [1, 2, 1, 3, 2] },  // 5 期排名
  { name: '球队 B', data: [3, 1, 2, 1, 4] },
  ...
]
```

---

## 关键规范

### 粗细

| 元素 | 值 |
| --- | --- |
| 线段描边 | **2px**（比基础折线 1.5px 略粗，强调排名变化） |
| 线型 | `solid` 实线 |

### 符号

| 元素 | 值 |
| --- | --- |
| 数据点形状 | `emptyCircle`（空心圆） |
| 数据点大小 | **4px** |
| 显示策略 | `showSymbol: true`、`showAllSymbol: 'auto'`（可能时显示所有；否则按标签间隔策略） |

### 端点标签（`endLabel`）

| 属性 | 说明 |
| --- | --- |
| 显示 | 默认关闭，启用后显示在折线右端 |
| 距离 | 距端点 **8px**（默认） |
| 数值动画 | `valueAnimation: true`（数值变化时滚动动画） |

端点标签常用于"显示当前排名 / 当前实体名"，让线条尽头一目了然。

### 选中（emphasis）态

| 行为 | 默认 |
| --- | --- |
| 数据点放大 | `scale: true`（选中点放大） |

---

## Y 轴（排名轴）

| 规则 | 说明 |
| --- | --- |
| 方向 | **倒序**——1 在顶部，N 在底部（排名 1 = 最高 = 最上方） |
| 范围 | 通常固定为 `[1, N]`，N = 实体总数 |
| 刻度 | 整数（1, 2, 3, ...） |

---

## 时间轴扩展（`withTimeline`）

排名线图可附加时间轴控件，支持回放：

| 配置 | 说明 |
| --- | --- |
| 范围（`range`） | 时间轴起 / 终值 |
| 最大范围（`maxRange`） | 整个时间序列的总长度 |
| 当前索引（`curIndex`） | 当前播放到的时间索引 |

启用后用户可拖动时间轴查看任意时点的排名状态。

---

## 动画

| 项 | 值 |
| --- | --- |
| 动画缓动 | `linear`（线性，便于跟踪连续变化） |
| 数据变化转场 | `universalTransition: { divideShape: 'clone' }`（数据切换时元素克隆过渡） |
| 渐进渲染（`progressive`） | 0（禁用，确保所有线同时呈现） |

---

## 颜色

每条线一个颜色，按 [tokens.md — 折线色板](../tokens.md) 顺序取色。颜色对应同一实体跨时间保持一致。

---

## 实现要点（库无关）

- **Y 轴倒序**：排名 1 必须在最上方——这是排名图的视觉惯例，搞反会误导。
- **端点标签是关键**：折线右端的标签让用户立刻知道"谁是当前第几名"，强烈推荐启用。
- **线宽略粗（2px）**：相比基础折线，排名线图线条更粗，强调"轨迹"感。
- **时间轴回放强化叙事**：用 `withTimeline` 让用户能"重播"排名变化过程，比静态图更有冲击力。
- **数据点 emptyCircle 不抢戏**：用空心圆而非实心，避免数据点喧宾夺主，重点在线条轨迹。

---

## Do & Don't

| | 规则 |
| --- | --- |
| ✅ | Y 轴倒序（排名 1 在顶） |
| ✅ | 线宽 2px，比基础折线略粗 |
| ✅ | 端点标签默认启用，标识当前排名 |
| ✅ | 时间轴回放增强叙事 |
| ❌ | 不要把 Y 轴正序（1 在底）——违反排名直觉 |
| ❌ | 不要用实心数据点——会抢线条的视觉重心 |
| ❌ | 不要让线宽 < 1.5px——排名变化不够突出 |
