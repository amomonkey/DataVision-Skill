# 数据标签（Data Label）

> 📌 **常用值已在 [themes/base.md（base 主题）](../themes/base.md) 内联**——画 base 主题图表时直接从那里取值。本文件是详细规范，仅在需要边界细节（动画分档、交互态、多端差异、配置选项等）时查阅。用了业务线主题（iFinD-PC / Ainvest / THS）请同时查 [themes/](../themes/) 下对应 delta 文件。

数据标签是直接显示在图形上的数值（柱顶 / 折点 / 段内 / 扇区外侧等）。各图表的数据标签可见性规则可能不同（见各 [chart 文档](../charts/)），本文件定义共享规范。

---

## 通用规范

| 属性           | 值                                   | Token                                      |
| ------------ | ----------------------------------- | ------------------------------------------ |
| 位置策略         | **通用原则**：避免遮挡核心数据；正值优先放图形上方、负值放下方；空间不足时按各图表规则偏移或隐藏。**具体位置值见各 [chart 文档](../charts/)**（柱状柱顶上方、折线折点旁、饼图扇区外侧 / 内、堆叠段内居中、横向条形柱右侧等） | —                                          |
| 字号           | 10px（移动端） / **12px**（PC）            | `font-size-xxs`（10）/ `font-size-extra-small`（12） |
| 字体           | THS Money font Medium               | `font-family-number`, `font-weight-medium` |
| 颜色           | **根据标签所处局部背景的对比度自动切换**：浅色背景（柱顶上方 / 图表空白区） → `color-text-primary`；深色填充内（堆叠段内 / 深色区段上 / 系列填充内） → `color-text-inverse-primary` | `color-text-primary` / `color-text-inverse-primary` |
| 默认显示         | 因图表而异（柱状默认显示 / 多折线默认隐藏 / 归一化堆叠默认隐藏），见各 [chart 文档](../charts/) | —                                          |
| 隐藏规则（常见）     | 折线图：数据点数量 > 5 时隐藏；PC 端：发生碰撞时隐藏       | —                                          |
| 0 值显示        | 正常显示「0」                             | —                                          |
| 无数据          | 不显示标签                               | —                                          |
