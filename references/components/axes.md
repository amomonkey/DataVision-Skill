# 坐标轴（Axes）

> 📌 **常用值已在 [themes/base.md（base 主题）](../themes/base.md) 内联**——画 base 主题图表时直接从那里取值。本文件是详细规范，仅在需要边界细节（动画分档、交互态、多端差异、配置选项等）时查阅。用了业务线主题（iFinD-PC / Ainvest / THS）请同时查 [themes/](../themes/) 下对应 delta 文件。

X 轴（类别 / 时间）和 Y 轴（数值）的标签、轴标题、刻度线规范。Y 轴标签的布局形式（避让 / 居中对齐）见 [layout.md § Y 轴标签布局](layout.md#y-轴标签布局)。轴标签 hover 高亮（胶囊样式）见 [tooltip.md § 光标线 / 高亮标签轴](tooltip.md#光标线--高亮标签轴)。

---

## X 轴标签

| 属性     | 值                                                                                                                                        | Token                                      |
| ------ | ---------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------ |
| 字号     | 10px（移动端） / **12px**（PC）                                                                                                                 | `font-size-xxs`（10）/ `font-size-extra-small`（12） |
| 行高     | 12px                                                                                                                                     | `line-height-super-small`                  |
| 字体     | THS Money font Medium                                                                                                                    | `font-family-number`, `font-weight-medium` |
| 颜色     | `rgba(0,0,0,0.6)`                                                                                           | `color-text-secondary`                     |
| 标签对齐   | 中间标签居中对齐；**首尾标签对齐取决于数据是否贴绘制区边缘**，规则见下方 ⚠️（本节为唯一权威）                                                                                       | —                                          |
| 左右安全间距 | 由绘制区两端预留，见 [layout.md § 尺寸与结构](layout.md#尺寸与结构)                                                                                          | —                                          |
| 上下间距   | 默认 3px（标签与网格线 / X 轴标题之间）；当 Y 轴使用形式 B（居中对齐，见 [layout.md § Y 轴标签布局](layout.md#y-轴标签布局)）时，最底部 Y 轴标签会向下溢出网格线，3px 不足以避让，需相应增大上间距以容纳半个 Y 轴标签字高 | —                                          |
| 碰撞处理   | 标签跟随数据动态显示；发生重叠时隐藏碰撞标签，首尾标签始终保留                                                                                                          | —                                          |

**X 轴时间标签示例**

| 周期 | 标签 |
| --- | --- |
| A 股日内 | `9:30`（左） / `11:30/13:00`（中） / `15:00`（右） |
| 日 / 周 / 月线 | `2022-01-01`（左） / `2022-06-15`（中） / `2023-01-01`（右） |

> A 股场景下不得省略 X 轴中间的午休分隔标签（`11:30/13:00`）。

> ⚠️ **首尾标签对齐 = 看数据是否贴绘制区边缘（本节为唯一权威；base 主题沿用此默认、不再复制规则，业务线主题只记 delta）**：
>
> - **首尾数据贴 grid 边缘**（折线 / 面积等 `boundaryGap: false`，首尾数据点正落在绘图区左 / 右沿）：首标签**左对齐**（锚点贴左、向右伸入），尾标签**右对齐**（锚点贴右、向左伸），中间居中——否则首尾以刻度居中会让外半截溢出容器被裁（症状：`2025-05`→`-05`、`2026-XX`→`26-`）；绘图区须为首尾预留「左右安全间距 24px」。
> - **首尾数据不贴边**（柱状族 / category 轴 `boundaryGap: true`，首尾落在 band 中心、距 grid 边有半 band 缓冲）：**全部标签居中对齐**，与柱 / 数据点对齐；**不要再加首左 / 尾右对齐**，否则首尾标签会离开 band 中心、与数据错位。
>
> 实现落地见 [echarts-implementation-hints.md 陷阱 11](../echarts-implementation-hints.md)。

## Y 轴标签

| 属性         | 值                                              | Token                                      |
| ---------- | ---------------------------------------------- | ------------------------------------------ |
| 字号         | 10px（移动端） / **12px**（PC）                       | `font-size-xxs`（10）/ `font-size-extra-small`（12） |
| 字体         | THS Money font Medium                          | `font-family-number`, `font-weight-medium` |
| 颜色         | `rgba(0,0,0,0.6)` | `color-text-secondary`                     |
| 主 Y 轴位置    | 图表左侧，右对齐                                       | —                                          |
| 副 Y 轴位置    | 图表右侧，左对齐                                       | —                                          |
| **默认显示位置** | **网格线内部**（标签贴近 Y 轴线内侧）                         | —                                          |
| 支持配置       | 网格线外部 + 左右对齐方式                                 | —                                          |
| 默认刻度数量     | 5 条（等间距）                                       | —                                          |
| 例外         | K 线图 Y 轴只显示 3 个价格刻度（顶 / 中 / 底）                 | —                                          |

## 双 Y 轴的启用条件

| 图表 | 双 Y 轴 |
| --- | --- |
| 折线图 1 条 | 单轴 |
| 折线图 2+ 条且量纲不同 | 双轴 |
| 折柱组合图 | 始终双轴（左柱右折线） |
| 红绿柱 + 折线 | 双轴 |
| K 线 + 折线 | 双轴 |

## 轴刻度线（Tick Marks）

| 属性 | 值 |
| --- | --- |
| 默认 | **不显示** |
| 启用后默认值 | 长度 3px，颜色 `color-visualization-divider`，粗细 1px |
| 支持配置 | 长度、粗细、颜色 |

## 轴标题（Axis Title）

| 属性     | 值                       | Token                     |
| ------ | ----------------------- | ------------------------- |
| 默认     | **不显示**（默认无轴标题）         | —                         |
| 位置     | 独立区域，**在图表外侧**，不影响图表内布局 | —                         |
| Y 轴标题  | 跟随 Y 轴，居左或居右显示（与轴在同一侧）  | —                         |
| X 轴标题  | 通常按最新一期数据的位置，**在右侧显示**  | —                         |
| 对齐     | 与最长的轴标签为准，**右对齐**       | —                         |
| 单位独立控制 | 标题单位 + 轴标题可分别显示 / 隐藏    | —                         |
| 字号     | 10px（移动端） / **12px**（PC） | `font-size-xxs`（10）/ `font-size-extra-small`（12） |
| 行高     | 12px (1.2)              | `line-height-super-small` |
| 字重     | Medium                  | `font-weight-medium`      |
| 颜色     | `rgba(0,0,0,0.6)` | `color-text-secondary`    |
| 上下间距   | 默认 2px                  | —                         |
